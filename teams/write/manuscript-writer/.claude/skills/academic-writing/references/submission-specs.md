# 투고 규격: APA7 · 분량 · 초록 · 익명화

원고를 1차 저널 SSCR(Social Science Computer Review, SAGE) 투고요건에 정렬한다. 양식 위반은 데스크리젝트 사유가 될 수 있다.

> 근거: paper-scout가 SSCR/SAGE 공식 author guidelines(2026-06-25 조회)에서 검증한 요건. 세부·갱신은 paper-scout의 `journal-submission-checklist` 스킬과 `.context/sscr-submission-checklist.md`가 정본. 저널이 바뀌면 해당 저널 요건으로 교체한다.

---

## 분량

- **총 10,000단어 한도(참고문헌 포함).** 다방법 원고에는 압박 → 보조분석은 supplementary로 이관 검토. **본문에서 임의 삭제하지 말고** 이관을 제안하고 보고 후 결정.
- 본문 분량을 카운트로 추적한다(번역 시 한글↔영문 팽창 주의: 한글 본문 → 영문 단어 수 추정으로 한도 초과 위험 사전 점검).

## 초록·키워드

- **초록 150-200단어.** 구조는 `section-moves.md`의 5문장(배경·문제·방법·결과·의의).
- **키워드 최소 5개.**

## 인용 스타일

- **APA 7th edition.** 본문 (저자, 연도), 참고문헌 APA7 형식.
- 인용 무결성은 `citation-evidence.md`(태깅·과대지지 금지).

## 익명화 (double-anonymized)

- **double-anonymized review.** 본문·메타데이터에서 식별정보 일체 제거.
- 저자명·소속·감사말·자기인용 식별 표현 제거.
- **헤더·푸터의 저자명도 제거**(흔한 누락 지점).
- Title Page는 분리하여 식별정보를 거기에만.

## 필수 진술 (Required statements)

- **AI/LLM 사용 공개** — [핵심] 본 원고는 LLM 코딩 131,792건을 사용하므로, 형식요건이자 방법 신뢰성의 핵심. Methods에 명시.
- 윤리(IRB 해당 여부 — PI 확인 필요, 2차·집계자료면 해당 없음 진술 가능).
- 동의: Participate/Publication 분리. 2차·집계자료라 Publication 동의는 해당 없음 진술 권장.
- 이해상충(COI), 연구비(Funding), 데이터 가용성(Data availability).
- Author Contributions: Acknowledgements 다음 위치(ICMJE).

## paper-scout 검증으로 정정된 항목 (오해 주의)

- **추천 심사자: 불요.** SSCR은 저자 추천을 명시 거부("we assign reviewers and cannot accept author recommendations"). 반대 심사자 항목도 없음. "추천 심사자 누락" 우려는 불요로 답한다.
- **ORCID: 필수 아님.** 강력 권장(수락 전 연결). 누락이 데스크리젝트 사유 아님.
- **투고·게재 비용: 무료**(구독지; 인쇄 컬러만 유료).
- **파일 형식: Word 선호, PDF 본문 불가.**
- **English editing: 권장(필수 아님).**

## 점검 체크리스트

- [ ] 총 단어 수 ≤ 10,000(참고문헌 포함). 초과 시 supplementary 이관 제안.
- [ ] 초록 150-200단어, 키워드 ≥ 5.
- [ ] APA7 본문 인용·참고문헌.
- [ ] double-anonymized: 본문·헤더·푸터·메타데이터 식별정보 제거, Title Page 분리.
- [ ] AI/LLM 사용 공개를 Methods에 명시.
- [ ] 윤리·동의·COI·Funding·데이터 가용성 진술 존재(해당 없음도 진술).
- [ ] Author Contributions 위치(Acknowledgements 다음).
- [ ] 파일 형식 Word.
