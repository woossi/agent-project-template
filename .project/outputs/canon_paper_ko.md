# 초록

본 문서는 `.project` canon의 claim-evidence-provenance 구조를 결정적으로 직렬화한 국문 학술 논문 골격이다. 본문의 주장 문장과 근거 부착은 canon record에서만 가져오며, 시스템은 새로운 claim 또는 evidence 내용을 생성하지 않는다.

# 1. 서론

본 절은 canon에 등록된 주장 구조를 논문 절 구조로 배열한다. 각 주장은 기존 canon의 문장을 사용하고, 근거는 각주로 부착한다.

# 2. 정당화 그래프

## 2.1. C001

무조건 모형에서 디지털 활용능력의 자치구간 분산 비중(ICC)은 0.49%로, 개인 수준 변량이 자치구 수준 변량보다 크게 나타났다.[^C001-E001]

[^C001-E001]: E001: ICC — 무조건 모형 자치구간 분산 비중, 값 0.49%; 출처 P001(model_result; D001; RUN001); 검증 stats-validator

## 2.2. C002

온라인 생활세계 텍스트 13만여 건을 행정동 단위로 매칭하고 LLM으로 분류하면, 행정지표가 포착하지 못하는 지역 단위 디지털 신호가 확인되었다.[^C002-E003]

[^C002-E003]: E003: 행정동 매칭 성공 LLM 분류 표본, 값 131,792; 출처 P003(pipeline_result; D002; RUN003); 검증 stats-validator

## 2.3. C003

통제 이전 모형을 기준으로, 저학력 집단은 대졸 기준집단과 비교할 때 디지털 이용 점수가 20점 이상 낮게 나타났다.[^C003-E001b]

[^C003-E001b]: E001b: 학력 중졸이하 효과(모형1, SE 0.770), 값 -22.618; 출처 P001b(model_result; D001; RUN002); 검증 stats-validator

## 2.4. C004

LLM 분류는 인간 코더와 비교할 때 관련성 정밀도 84.7%, 주 차원 κ=0.673의 일치도를 보였다.[^C004-E007][^C004-E008]

[^C004-E007]: E007: UMC 관련성 정밀도(LLM Y→인간 확인), 값 84.7%; 출처 P007(validation_result; D002; RUN003); 검증 stats-validator
[^C004-E008]: E008: 기기(DEV) 차원 F1 — 6차원 최저(해석 게이트), 값 0.51; 출처 P008(validation_result; D002; RUN003); 검증 stats-validator

## 2.5. C005

본 분석의 개인 수준 모형은 서울서베이 2023–2024년 통합 표본 10,000명을 대상으로 한다.[^C005-E002]

[^C005-E002]: E002: 서울서베이 HLM 표본(2023–2024 통합), 값 10,000명; 출처 P002(data_spec; D001; RUN002); 검증 stats-validator

## 2.6. C006

상호작용을 포함한 모형을 기준으로, 65세 이상 집단은 65세 미만과 비교할 때 디지털 이용 점수가 약 9점 낮게 나타났다.[^C006-E006]

[^C006-E006]: E006: 65세 이상 효과(모형4, SE 0.467), 값 -9.303; 출처 P006(model_result; D001; RUN002); 검증 stats-validator

## 2.7. C007

UMC 종합 지수는 서울내 상대 척도에서 중랑구 0.279(최저)와 비교할 때 서초구 0.695(최고)로, 자치구 간 차이를 보였다.[^C007-E009]

[^C007-E009]: E009: 종합 지수 최저(중랑구, 서울내 min-max), 값 0.279; 출처 P009(index_result; D003; RUN005); 검증 stats-validator

## 2.8. C008

행정동 단위 온라인 텍스트에서 산출한 중랑구 연결품질의 차원내 표준화 z_shift는 +1.68로, 같은 차원 내 다른 자치구와 비교할 때 상위 우선검토 후보로 나타났다.[^C008-E010][^C008-E003]

[^C008-E010]: E010: 중랑 연결품질 z_shift(★주정본, 차원내 표준화), 값 +1.68; 출처 P010(model_result; D003; RUN005); 검증 stats-validator
[^C008-E003]: E003: 행정동 매칭 성공 LLM 분류 표본, 값 131,792; 출처 P003(pipeline_result; D002; RUN003); 검증 stats-validator

## 2.9. C009

정보차단 멀티에이전트 역행추론은 표적 3구의 게시글 223건에서 지역 가설 772개를 도출하였다.[^C009-E004][^C009-E005]

[^C009-E004]: E004: 표적 3구 단계 E 판정 게시글, 값 223건; 출처 P004(pipeline_result; D002; RUN004); 검증 stats-validator
[^C009-E005]: E005: 223건에서 생성된 지역 가설 총수, 값 772; 출처 P005(model_result; D002; RUN004); 검증 stats-validator

# 3. 결론

위 구조는 canon의 현재 claim 위상 순서와 evidence 부착 상태를 반영한다. 추가 산문화는 이 골격의 claim 순서와 evidence 각주를 변경하지 않는 범위에서만 허용된다.
