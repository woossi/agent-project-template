# R6 종합 검수 지시 (quality-reviewer)

mw가 R&R 2차 대응분(R1 재현성·R2 데이터윤리·R3 과대주장·R4 구조재작성·제목)을 완결했다(빌드 PASS 24p). R6 종합 검수를 수행하라 — R&R 재투고 직전 최종 품질 게이트다. manuscript-review-gate 스킬 적용.

## 검수 항목

1. **관통논리 E축**(이미 편입): (1)긍정선언=절차/설계 유지됐나, (3)선제사과 항변 제거 유지, (2)진짜한계 §5 보존. 특히 R3 과대주장 하향이 (2)진짜한계를 과삭제 안 했나.

2. **R3 과소헤지**(중점): mw가 개념어를 공격적으로 하향했다(예: "인과기제 규명" 을 "구조적 기제 탐색"으로). 반대로 검증된 발견(중랑 CQ 단정 = 강한 근거)까지 과하게 약화시켜 식별력을 깎았는지 점검. sv 자치구 차등(중랑 단정 / 강북 이원 / 노원 가설화)이 정확히 반영됐나.

3. **R4 구조**: 서론 5단 재배열이 명료한가, RQ별 핵심결과가 빠르게 파악되나, 방법방어 부록 D 이동이 정합한가(본문에 참조구만 남았나), §2 이론장 45% 축소가 핵심 방법정당화("왜 행정지표와 플랫폼텍스트를 직접 동일시하지 않는가")를 훼손 안 했나.

4. **R1 재현성**: 부록 명세표·프롬프트 발췌가 자체완결적인가(외부저장소 의존 제거됐나). 223 표본 재현성을 과장(재현가능 단정)하지 않았나 — 절차 재현성만 단정해야 하고, 표본 선정(층화표집 ID 유실)은 미보존 사실만 명시해야 함.

5. **R2 데이터윤리**: §3.3 윤리절(subsubsec:ethics)이 방법 핵심조건으로 충분한가, ToS 톤이 위반 단정도 회피도 아닌 정직 수준인가(danggeun 06_tos_privacy_review.md 근거와 정합).

6. **제목**: "서울의 디지털 포용 우선순위: 자치구 간 장소 민감적 측정" — 간결한가, 도시계획판 적합한가.
