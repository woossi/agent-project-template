## 1. 관통 원리 (한 층 추상화 — 이것이 단일 논리)
이 논문의 모든 "한정·단서·해명"은 **세 종류뿐**이고, 종류마다 동작이 **결정**된다.

| 종류 | 정의 | 동작 |
|---|---|---|
| **(1) 설계가 요구한 것** | 목적·단위·척도·정보차단이 연역적으로 잠근 귀결. (min-max·자치구단위·정보차단 설계·예측오차 우선탐색) | **긍정 선언** — 방어가 아니라 주장. "X는 목적이 요구한 설계 귀결이다." |
| **(2) 데이터가 말할 수 없는 것** | 선택편향 비식별성·통신사(SKT)대리·낮은 ICC 검정력·본문한정(댓글부재). | **사실로 1회 적시** — §5 한 곳. 항변이 아니라 경계. |
| **(3) 비판 선제사과 항변** | (1)을 (2)인 척 위장한 것. "존재론 동일성 아님·발견법으로만·등가 아님" 반복, "탐색적 윤곽으로만" 반복, "차선 아님" 부정경유. | **삭제** — (1)을 긍정선언으로 바꾸면 (3)은 불필요해진다. |

핵심: **A는 (1)로 승격. B·C는 (3)으로 식별→삭제. 진짜 (2)만 남긴다.**
즉 사용자의 "A=설계 1층 / B·C=절제 2층"의 운영형: 설계층은 주장하고, 절제층은 (진짜 한계만 빼고) 지운다.

## 2. 전개 좌표 (원리가 글의 어디서 무슨 형태로 나타나는가)
관통 논리는 추상명사로만 주면 매 절에서 방어로 회귀한다. 위치·형태를 좌표로 박는다.

- **서론(intro)**: 설계 사슬(맥락→목적→단위→척도→정보차단)을 **1회 긍정 선언**. 가교는 "발견법으로 빌린다"를 *한 번만* 담백하게(반복 항변 금지). [현재 intro:29·45 = 항변 2회 → 1회로]
- **이론(ch2)**: 가교를 **1회만 논증**(fletcher2017로 자의성 갭→우리 기여 동기). 존재론 동일성 부인 문장(21·23·25·27의 반복) → 1개 논증으로 압축. "통찰 의존 완전제거 아님"은 (2)진짜한계라 1회 보존 가능. [현재 ch2 = 항변 4회 → 1회 논증으로]
- **분석모형(model §3.1.1)**: min-max를 **설계 정당화로 1회**(서론 사슬 참조). "차선/대용 아님" 부정경유 삭제, "우선순위=순서문제→상대척도=귀결" 긍정형. [model:40]
- **결과(results)**: 전제 분산반복 금지. **짧은 참조구로만**("우선순위 배분 위한 자치구 단위 상대 척도상"). 자료뒷받침 발견은 단정. [results 다수의 "탐색적으로만" 헤지 제거]
- **결론(ch4 §5.1·요약)**: 설계논리 **회수** — 측정→정책 우선순위 이행이 자의적 해석 아니라 목적·단위·척도가 잠긴 설계 귀결. [ch4:43 = 이미 좋음. ch4:34 = 항변덩어리라 (3)삭제 대상]

## 3. 경계 (지우지 말 것 — 가장 자주 틀리는 지점)
- **(2)진짜한계는 절대 단정화 금지·삭제 금지.** 선택편향 비식별성(ch4:30 첫째), SKT 대리(둘째), min-max 적용범위가 절대충분성 판단 아님(셋째), 검정력(다섯째), 동일출처 제외 대가(여섯째) — 전부 §5에 **사실로 1회씩 보존**. 이걸 "설계 귀결"로 긍정선언하면 논문이 한계를 숨기는 더 나쁜 상태가 된다.
